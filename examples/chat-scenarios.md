# English Coach — Practical Chat Scenarios

This file gives concrete, copy-pasteable ways to use English Coach in daily life.

## 1. Beginner: simple question with correction

You type:

```text
/english-coach
how can i practice english every day if i am busy?
```

What you should expect:
- Claude answers the actual question first
- then corrects capitalization / grammar / expression
- then teaches one small phrase or grammar pattern

Why this is useful:
- you do not need to switch into a separate “study session”
- you improve English while solving a real problem

## 2. Beginner: ask for gentle corrections only

```text
/english-coach
Please answer normally, but keep the correction section very short because I'm a beginner.
I want to write english every day but sometimes i dont know what to say.
```

Best for:
- people who feel overwhelmed by too many corrections
- daily low-pressure practice

## 3. Intermediate: ask for more natural phrasing

```text
/english-coach
I'm preparing for work conversations. Please focus more on natural phrasing than basic grammar.
How can I say this more naturally: "I want to ask if you have time join this meeting"?
```

Best for:
- workplace English
- speaking more naturally instead of only “being correct”

## 4. Intermediate: ask for pattern tracking

```text
/english-coach
Please watch for my recurring mistakes. If I repeat the same grammar problem, call it out clearly.
Today I want to practice writing status updates for work.
```

Best for:
- users who want faster improvement
- identifying habits like article mistakes, tense drift, or awkward word choice

## 5. Advanced: ask for concise polish

```text
/english-coach
Please keep the correction section concise and focus on tone, naturalness, and precision.
Help me improve this sentence for a professional setting:
"I think this plan is good, but maybe we should discuss more deeply."
```

Best for:
- users who already have decent grammar
- business writing and polished communication

## 6. Real workflow: use it while asking technical questions

```text
/english-coach
Can you explain why this Python function returns None sometimes?
```

Why this is useful:
- English Coach still answers the technical question
- you get English improvement without pausing your real work

## 7. Real workflow: use it while preparing messages

```text
/english-coach
Please help me write a short message to my manager:
"I finish the task but I need one more day to test all details."
```

Best for:
- email drafts
- Slack / Discord / Teams messages
- polite professional English

## 8. Ask for one specific teaching style

If you want the skill to behave differently, tell it directly.

Examples:

```text
Focus on grammar only.
```

```text
Focus on natural spoken English.
```

```text
Please always give one reusable phrase of the day.
```

```text
Correct only the 3 most important mistakes.
```

## 9. Good habit: keep the prompt stable for a few days

A practical way to get more value:
- keep using the same correction preference for several days
- ask the coach to watch recurring patterns
- compare whether the same mistakes keep appearing

Example:

```text
For the next few messages, please keep tracking my recurring mistakes and focus on business English.
```

## 10. When English Coach is NOT the best fit

This skill is less ideal when:
- you want full textbook-style lessons
- you want zero corrections and only a normal answer
- you are reviewing long formal documents and want line-by-line editing

In those cases, use a more specialized writing or editing workflow.

## Suggested first-use sequence

If you are trying the skill for the first time, use this order:

1. Ask one normal question in English
2. Ask the coach to reduce or increase correction intensity
3. Ask it to focus on your real goal: speaking, writing, work English, or natural phrasing
4. Reuse the same mode for a week

That gives you a much better signal than one-off testing.
