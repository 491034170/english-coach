# English Coach

Навык для [Claude Code](https://docs.anthropic.com/en/docs/claude-code), который превращает каждый разговор в урок английского.

**Другие языки:**

## Что делает

При активации Claude делает три вещи в каждом ответе:

1. **Отвечает на ваш вопрос** — сначала выполняет основную работу
2. **Исправляет ваш английский** — грамматика, орфография, лексика, пунктуация, выражения
3. **Учит чему-то новому** — полезные фразы, грамматические советы, более естественные способы выражения

Создан для **неносителей английского языка**, которые хотят улучшить свой английский через ежедневное общение, а не через упражнения из учебника.

## Возможности

- **Исправления по категориям**: Орфография / Грамматика / Выбор слов / Пунктуация / Выражение
- **Адаптация к уровню**: для начинающих — базовые исправления, для продвинутых — естественность речи
- **Отслеживание повторяющихся ошибок**: если ошибка повторяется, она отмечается как паттерн
- **Ротация обучающего контента**: идиомы, грамматические советы, продвинутые перефразировки, типичные ошибки русскоязычных
- **Дружелюбный тон**: как отзывчивый коллега, а не строгий учитель

## Установка

```bash
npx skills add 491034170/english-coach -a claude-code -g -y
```

## Использование

В Claude Code введите:

```
/english-coach
```

Затем просто общайтесь на английском как обычно. Навык автоматически исправляет и обучает.

## Пример

**Вы пишете:**
> I live in Moscow since 10 years and I am work as programmer

**Claude отвечает:**

*(отвечает на вопрос)*

---

**English Corrections:**

> ~~I live in Moscow since 10 years~~ → **I've been living in Moscow for 10 years**
>
> **[Грамматика]** "live" → "have been living" — для действий, начатых в прошлом и продолжающихся сейчас, используется Present Perfect Continuous
> **[Грамматика]** "since 10 years" → "for 10 years" — "since" для конкретных дат, "for" для длительности
>
> ~~I am work~~ → **I work**
>
> **[Грамматика]** "I am work" → "I work" — для профессии используется Present Simple без "am"

**Фраза дня:** "get the hang of" — освоиться, разобраться в чём-то.
*Пример: I'm starting to get the hang of English articles. (Я начинаю разбираться в английских артиклях.)*
